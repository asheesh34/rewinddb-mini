# RewindDB Mini

Git-like version control for PostgreSQL databases.

## Overview

RewindDB Mini tracks database changes and provides foundations for history, comparison, and rollback workflows.

## Modules

- Backend: Spring Boot 3, Java 21, PostgreSQL, Flyway
- Frontend: React, TypeScript, TailwindCSS
- Infrastructure: Docker, Docker Compose, GitHub Actions
